# Data Distribution and Visualization

## Overview
This section presents the statistical distribution characteristics of key variables and outlines the visualization strategy implemented in Power BI dashboards. A professional color palette has been applied consistently across all visualizations to ensure clarity, accessibility, and brand coherence.

## Color Palette & Design Standards

### Primary Color Scheme
A sophisticated four-color palette was selected to balance professional aesthetics with data readability:

| Color Code | Color Name | Usage | Visual Impact |
|------------|------------|-------|---------------|
| `#F4EDD3` | Warm Beige | Background, neutral elements | Soft, reduces eye strain |
| `#1D546C` | Teal Blue | Primary KPIs, positive metrics | Professional, trustworthy |
| `#1A3D64` | Navy Blue | Secondary data series, comparisons | Strong contrast, authoritative |
| `#0C2B4E` | Dark Navy | Text, negative metrics, emphasis | Maximum readability |

### Design Principles Applied
- **High Contrast**: Dark navy text (#0C2B4E) on cream background (#F4EDD3) ensures 7:1 contrast ratio (WCAG AAA compliance)
- **Sequential Scaling**: Blue gradient (#1D546C â†’ #1A3D64 â†’ #0C2B4E) represents low-to-high data ranges
- **Categorical Distinction**: Teal (#1D546C) for primary focus, Navy (#1A3D64) for secondary categories
- **Status Indicators**: 
  - Positive/Good: Teal (#1D546C)
  - Neutral/Warning: Beige accents
  - Negative/Critical: Dark Navy (#0C2B4E)

## Visualization Best Practices Implemented

### Accessibility
âœ… **Color Blind Safe**: Blue-beige palette tested for deuteranopia and protanopia  
âœ… **High Contrast**: 7:1 ratio between text and background  
âœ… **Alternative Text**: All visuals include descriptive alt-text  
âœ… **Legend Clarity**: Text labels supplement color coding  

### Clarity
âœ… **Minimal Clutter**: Maximum 4 visuals per dashboard screen  
âœ… **Consistent Layout**: Grid-based alignment  
âœ… **Data-Ink Ratio**: Removed unnecessary gridlines and decorations  
âœ… **Hierarchy**: Largest visuals = most important insights  

### Interactivity
âœ… **Cross-Filtering**: Click any chart element to filter others  
âœ… **Tooltips**: Hover reveals detailed metrics  
âœ… **Drill-Through**: Double-click for granular analysis  
âœ… **Reset Button**: Clear all filters quickly  

### Performance
âœ… **Query Optimization**: Star schema enables sub-second load times  
âœ… **Aggregated Tables**: Pre-calculated totals for common queries  
âœ… **Incremental Refresh**: Only new data loads on dashboard refresh
